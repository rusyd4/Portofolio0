Saya memiliki pemahaman seperti merancang user interface menggunakan React.js dan React Native, pengembangan backend dengan Node.js dan Express.js, serta pengelolaan database seperti MongoDB dan PostgreSQL, sekaligus mengeksplorasi sistem embedded dan Internet of Things (IoT) dengan pendekatan real-time menggunakan ESP32, Arduino dan FreeRTOS.
